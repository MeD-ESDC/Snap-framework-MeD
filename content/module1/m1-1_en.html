Standard Components

Using Components

There are several different components available for formatting your content. The Snap! Framework is built on the WET template and utilizes the widgets that have been created at part of that template. To align the look and feel of courses built using the Snap! framework with those built in Adapt, we will be using a 2-column layout. Components can either be full-width or half-width of the block. Within each block the maximum number of components is limited to 2. This allows the using to focus on one block of content at a time, similar to a slide in Captivate.

Explore the available components for your layout ideas

Callouts

Use Callouts when you want to pull out or emphasis a text component. Callouts are styled differently than regular text components to draw the learner's focus to special information.

Video

For video, the multimedia player leverages the native HTML5 video tag. The MPEG 4 format (H264 codec) is the minimum requirement for video because it is the standard video format on many browsers. The multimedia player's timeline relies on the HTML5 progress element and uses a polyfill when the element is not supported.
