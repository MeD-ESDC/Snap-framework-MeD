Headings (h3)

HTML defines six levels of headings. The heading elements are H1, H2, H3, H4, H5, and H6 with H1 being the highest (or most important) level and H6 the least.

Headings and Accessibility (h4)

Using descriptive headings is helpful for all users but it is especially beneficial to people who use accessible technologies. Properly written headings allow users to quickly scan and navigate content on a page and help to determine the structure of the information being presented.

Course Navigation

In the Snap! framework we have several methods of navigating the course. There is back button in the menu bar, a sitemap in the menu bar, and Previous and Next buttons in the navigation bar at the end of each page. The navigation bar is a component within the last block. Use the navigation block to introduce the next topic or page.
